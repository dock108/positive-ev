graph TD
    Start[Start] --> PlayerHand{Player Hand Type?}
    
    %% Splitting Logic
    PlayerHand --> |Pair| PairType{What Pair?}
    PairType --> |Aces| SplitAces[Always Split]
    PairType --> |8s| Split8s[Always Split]
    PairType --> |2s or 3s| Split23[Split if Dealer 2-7, Else Hit]
    PairType --> |6s| Split6[Split if Dealer 2-6, Else Hit]
    PairType --> |7s| Split7[Split if Dealer 2-7, Else Hit]
    PairType --> |9s| Split9[Split if Dealer 2-9 Except 7, Else Stand]
    PairType --> |10s| NoSplit10[Never Split]
    PairType --> |4s or 5s| NoSplitLow[Never Split]
    
    %% Soft Hand Logic
    PlayerHand --> |Soft Total| SoftHand{What Soft Total?}
    SoftHand --> |13-14| Soft13[Double if Dealer 5-6, Else Hit]
    SoftHand --> |15-16| Soft15[Double if Dealer 4-6, Else Hit]
    SoftHand --> |17| Soft17[Double if Dealer 3-6, Else Hit]
    SoftHand --> |18| Soft18[Double if Dealer 3-6, Stand if 2,7-8, Else Hit]
    SoftHand --> |19+| Soft19[Always Stand]
    
    %% Hard Hand Logic
    PlayerHand --> |Hard Total| HardHand{What Hard Total?}
    HardHand --> |4-8| HardLow[Always Hit]
    HardHand --> |9| Hard9[Double if Dealer 3-6, Else Hit]
    HardHand --> |10| Hard10[Double if Dealer 2-9, Else Hit]
    HardHand --> |11| Hard11[Double if Dealer 2-10, Else Hit]
    HardHand --> |12| Hard12[Stand if Dealer 4-6, Else Hit]
    HardHand --> |13-16| Hard13[Stand if Dealer 2-6, Else Hit]
    HardHand --> |17+| Hard17[Always Stand]
    
    %% Insurance Logic
    PlayerHand --> |Dealer Ace| Insurance{Insurance Offered?}
    Insurance --> |Yes| Decline[Do Not Take Insurance]
    Insurance --> |No| Continue[Continue Play]
